# GetHeight


```c++
unsigned long GetHeight(void)
```

## Parameters
None.

## Return value
The height of the active display.

## Description
Retrieves the height of the active display, this can be the backbuffer or the OpenGL window.

## Version changes
- EDGELIB 3.00: Added this function to the SDK
