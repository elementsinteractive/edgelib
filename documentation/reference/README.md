# Reference Guide

EDGELIB is the middleware itself, containing all modules and interface APIs. Applications hook up on EDGELIB using the entry and wrapper functions. After that, applications are able to access all features on each supported platform.

This API reference describes all module methods, which parameters can be passed to them, the possible return values and a detailed description. Also, a concise history log shows the changes per version.
