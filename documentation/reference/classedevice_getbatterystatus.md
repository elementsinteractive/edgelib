# GetBatteryStatus


```c++
unsigned char static GetBatteryStatus(void)
```

## Parameters
None.

## Return value
Returns the current [status](classedevice_definitions.md) of the battery.

## Description
Read the status of the battery.

## Note
Currently support for Windows Mobile, Symbian and GP2X.
