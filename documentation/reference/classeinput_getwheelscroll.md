# GetWheelScroll


```c++
long GetWheelScroll(void)
```

## Parameters
None.

## Return value
The scrollwheel position.

## Description
Returns the direction of the wheel device. After calling this function, the direction is reset to 0.

## Version changes
- EDGELIB 2.70: Added this function to the SDK
