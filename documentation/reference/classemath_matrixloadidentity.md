# MatrixLoadIdentity


```c++
void static MatrixLoadIdentity(long *matrix)
```

## Parameters
### matrix (out)
The matrix to fill.

## Return value
None.

## Description
Load the identity matrix and set it to the parameter. Matrixes contain 20:12 fixed point values.

## Version changes
- EDGELIB 3.00: Added this function to the SDK
