# SetChr


```c++
void SetChr(unsigned long index, WCHAR chr)
```

## Parameters
### index (in)
The position of the character to change.

### chr (in)
The new character.

## Return value
None.

## Description
Change a single character in the string array. For example, using index 4 and chr 'A' changes the fifth character to A.
