# Resume


```c++
void Resume(void) 
void Resume(unsigned long ntime)
```

## Parameters
### ntime (in)
The current system time tick count.

## Return value
None.

## Description
Resumes the timer. When no argument is given, it will retrieve the time tick count internally.
