# SetColorKey


```c++
void SetColorKey(COLORREF col = E_RGBX(0, 0, 0))
```

## Parameters
### col (in)
The color that should be specified as the internal color key (transparent pixel).

## Return value
None.

## Description
Specify the color key, colors that are treated as transparent pixels.
