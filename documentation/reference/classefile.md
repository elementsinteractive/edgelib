# ClassEFile

This is a file module containing functions to open and create files, read and write to files and to read folders. It's also possible to read and write Edge pack files.

Always add the current file path when opening or writing files. The current file path can be found inside the [EDGEDATA](ecd_edgedata.md) structure.
