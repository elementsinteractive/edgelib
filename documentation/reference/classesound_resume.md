# Resume


```c++
void Resume(void)
```

## Parameters
None.

## Return value
None.

## Description
Resumes sound output. This is called automatically by EDGELIB when the application is restored from minimized state.

## Version changes
- EDGELIB 2.95: Added this function to the SDK
