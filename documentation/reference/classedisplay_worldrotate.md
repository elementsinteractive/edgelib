# WorldRotate


```c++
void WorldRotate(long xr, long yr, long zr)
```

## Parameters
### xr, yr, zr (in)
The amount of rotation, a value between 0 and 4096.

## Return value
None.

## Description
Rotates world coordinates for the EDGELIB and OpenGL 3D renderer.

## Version changes
- EDGELIB 3.00: Added this function to the SDK
