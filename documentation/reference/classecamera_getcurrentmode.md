# GetCurrentMode


```c++
void GetCurrentMode(E_CAMERAMODE & info);
```

## Parameters
### info (out)
the current [settings](classecamera_structures.md) for the camera

## Return value
none

## Description
Returns a mode description corresponding to the current camera configuration.
