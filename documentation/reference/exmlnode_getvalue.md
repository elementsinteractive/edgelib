# GetValue


```c++
WCHAR *GetValue(void)
```

## Parameters
None.

## Return value
A pointer to a string containing the value of the node.

## Description
Returns the node value. If the node has no value, it returns an empty string.

## Version changes
- EDGELIB 3.80: Added this function to the EDGELIB SDK
