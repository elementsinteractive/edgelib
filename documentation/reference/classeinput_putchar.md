# PutChar


```c++
void PutChar(WCHAR chr, unsigned long scancode = 0)
```

## Parameters
### chr (out)
A string character.

### scancode
The scancode of the string character. This parameter is optional.

## Return value
None.

## Description
Adds a character to the character buffer.

## Version changes
- EDGELIB 2.70: Added this function to the SDK
