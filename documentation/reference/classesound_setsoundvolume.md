# SetSoundVolume


```c++
void SetSoundVolume(unsigned char newvolume)
```

## Parameters
### newvolume (in)
The new volume, a value between 0 (silence) and 255 (loud).

## Return value
None.

## Description
Changes the volume of sound effects.

- EDGELIB 2.95: Added this function to the SDK
