# Start


```c++
MAINRET Start(void)
```

## Parameters
None.

## Return value
This is the value that should be returned by the program entry function. Call `Start()` within `EdgeMain`.

## Description
Call this function after the construction of `ClassMain` within `EdgeMain`. Use it to run the engine.
