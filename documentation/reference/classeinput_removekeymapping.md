# RemoveKeyMapping


```c++
bool RemoveKeyMapping(unsigned long nr)
```

## Parameters
### nr (in)
The key to remove from the mapping list.

## Return value
Returns true if the key is successfully unmapped, otherwise returns false.

## Description
Removes a mapping.

- EDGELIB 2.70: Added this function to the SDK
