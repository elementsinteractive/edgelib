# GetWorldMatrix


```c++
void GetWorldMatrix(long *matrix)
```

## Parameters
### *matrix (out)
A 20:12 fixed point matrix which is filled with the world matrix values.

## Return value
None.

## Description
Retrieve the world transformation matrix.
