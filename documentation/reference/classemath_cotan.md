# Cotan


```c++
long Cotan(unsigned long index)
```

## Parameters
### index (in)
The value to pass through the cotangent calculator.

## Return value
The cotangent value.

## Description
Calculate the cotangent value. The value is returned as fixed point(16:16).

- EDGELIB 3.00: Added this function to the SDK
