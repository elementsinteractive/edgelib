# Rotate


```c++
void Rotate(long xr, long yr, long zr)
```

## Parameters
### xr, yr, zr (in)
Rotate the model over the given axis. A value between 0 and 4096 represents a whole circle.

## Return value
None.

## Description
Rotate the model around several axes.
