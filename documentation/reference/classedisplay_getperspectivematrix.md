# GetPerspectiveMatrix


```c++
void GetPerspectiveMatrix(long *matrix)
```

## Parameters
### *matrix (out)
A 20:12 fixed point matrix which is filled with the perspective matrix values.

## Return value
None.

## Description
Retrieve the perspective transformation matrix.
