# KeyHold


```c++
bool KeyHold(unsigned char action)
```

## Parameters
### action (in)
The action to check.

## Return value
Returns true if any keys mapped to the given action is being held.

## Description
Check if any keys are held related to the specified action.

## Version changes
- EDGELIB 2.70: Added this function to the SDK
