# E2DSurface

A 2D surface is a part of allocated memory containing a rectangle of pixels. A surface can have different sizes. 
Use surfaces to draw pixels to other surfaces or the display buffer. A 2D surface can also be attached to a [3D surface](e3dsurface.md) as a texture.
