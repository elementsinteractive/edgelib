# Quit


```c++
void Quit(void)
```

## Parameters
None.

## Return value
None.

## Description
Calling this function will shutdown the engine.

## Note
This function is ignored by the iPhone build.

## Version changes
- EDGELIB 3.98: Removed support for iPhone as it was using the non-public API "terminate".
