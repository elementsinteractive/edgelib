# operator WCHAR*


```c++
operator WCHAR*()
```

## Parameters
None.

## Return value
A zero terminated Unicode string.

## Description
Allows the string object to be typecasted to WCHAR * to return a zero terminated Unicode string.

## Version changes
- EDGELIB 3.30: Added this operator to the EDGELIB SDK
