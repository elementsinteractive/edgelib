# SetClipper


```c++
void SetClipper(RECT *rc)
```

## Parameters
### *rc (in)
A pointer to a rectangle structure specifying the area to clip.

## Return value
None.

## Description
Sets a clipper to OpenGL or the backbuffer.

- EDGELIB 3.97: Added this function to the SDK
