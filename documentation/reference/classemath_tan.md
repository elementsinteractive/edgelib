# Tan


```c++
long Tan(unsigned long index)
```

## Parameters
### index (in)
The value to pass through the tangent calculator.

## Return value
The tangent value.

## Description
Calculate the tangent value. The value is returned as fixed point(16:16).

- EDGELIB 3.00: Added this function to the EDGELIB SDK
