# GetName


```c++
WCHAR *GetName(void)
```

## Parameters
None.

## Return value
A pointer to a string containing the name of the node.

## Description
Returns the node name. If the node has no name, it will return an empty string.

## Version changes
- EDGELIB 3.30: Added this function to the EDGELIB SDK
