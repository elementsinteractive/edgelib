# DeAlloc


```c++
void static DeAlloc(void *ptr)
```

## Parameters
### *ptr (in)
The pointer to the memory to deallocate. A NULL pointer can't be deallocated and will be ignored.

## Return value
None.

## Description
Deallocate previously allocated memory.
