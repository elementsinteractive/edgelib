# SetMatrix


```c++
void SetMatrix(long *newmatrix)
```

## Parameters
### newmatrix (in)
The matrix to set to this surface. The matrix should contain 20:12 fixed point values

## Return value
None.

## Description
Manually set a matrix to the surface.
