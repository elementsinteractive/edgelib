# DeAllocList


```c++
void DeAllocList(void *ptr)
```

## Parameters
### *ptr (in)
The pointer to the memory to deallocate.

## Return value
None.

## Description
Deallocate previously allocated memory. Use matching allocate and deallocation functions: alloc/dealloc and alloclist/dealloclist.
