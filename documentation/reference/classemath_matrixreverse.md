# MatrixReverse


```c++
void static MatrixReverse(long *matrix)
```

## Parameters
### matrix (in/out)
The matrix to reverse.

## Return value
None.

## Description
Reverse the operations for the specified matrix. Matrixes contain 20:12 fixed point values.

- EDGELIB 3.00: Added this function to the SDK
