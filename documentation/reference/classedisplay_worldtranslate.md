# WorldTranslate


```c++
void WorldTranslate(long x, long y, long z)
```

## Parameters
### x, y, z (in)
The amount of translation.

## Return value
None.

## Description
Translates world coordinates for the EDGELIB and OpenGL 3D renderer.
