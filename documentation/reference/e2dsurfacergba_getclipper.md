# GetClipper


```c++
void GetClipper(RECT *rc)
```

## Parameters
### *rc (out)
A pointer to a [rectangle structure](ref_globalstructures.md) to receive the clipper rectangle.

## Return value
None.

## Description
Receive the current clipper rectangle.
