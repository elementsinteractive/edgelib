# CheckRepeat


```c++
void CheckRepeat(unsigned long timedelta)
```

## Parameters
### timedelta (in)
The difference in time.

## Return value
None.

## Description
For input objects not part of [ECD](ecd.md): call this function manually each frame for generating key repeat events.
