# AutoSetColorKey


```c++
void AutoSetColorKey(long x = 0, long y = 0)
```

## Parameters
### x, y
The coordinate of the pixel.

## Return value
None.

## Description
Specify the color key, colors that are treated as transparent pixels. The colorkey will be taken from the given pixel coordinate.
