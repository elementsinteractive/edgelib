# Get


```c++
WCHAR *Get(void)
```

## Parameters
None.

## Return value
A zero terminated unicode string.

## Description
Call this function to get the unicode string to pass to one of the other EDGELIB functions.

## Version changes
- EDGELIB 3.30: Added this function to the EDGELIB SDK
