# PackValid


```c++
bool PackValid(void)
```

## Parameters
None.

## Return value
Returns true if the file is opened and identified as an EDGELIB resource pack (.epk), otherwise returns false.

## Description
Check if an opened file is an Edge pack.

## Version changes
- EDGELIB 2.60: Added this function to the SDK
