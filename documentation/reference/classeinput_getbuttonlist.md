# GetButtonList


```c++
void GetButtonList(EBUTTONLIST *blist)
```

## Parameters
### * blist (out)
A pointer to an [EBUTTONLIST](ecd_ebuttonlist.md) structure containing key mappings.

## Return value
None.

## Description
Fill a button list structure with key mappings.
