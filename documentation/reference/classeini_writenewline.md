# WriteNewline


```c++
bool WriteNewline(void)
```

## Parameters
None.

## Return value
Return true if a newline could be written to the INI file successfully, otherwise return false.

## Description
Write a newline ("\r\n") to an opened INI file.

## Version changes
- EDGELIB 2.60: Added this function to the SDK
