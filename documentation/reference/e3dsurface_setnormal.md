# SetNormal


```c++
void SetNormal(unsigned long index, signed char x, signed char y, signed char z)
```

## Parameters
### index (in)
The specified 3D vertex point.

### x, y, z (in)
The normal of the vertex point.

## Return value
None.

## Description
Define vertex normals manually.
