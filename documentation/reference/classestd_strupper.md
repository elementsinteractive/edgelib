# StrUpper


```c++
char static *StrUpper(char *str) 
WCHAR static *StrUpper(WCHAR *str)
```

## Parameters
### *str (in/out)
The string to scan

## Return value
A pointer to the parameter *str.

## Description
Changes a string to uppercase.
