# GetLastKey


```c++
unsigned long GetLastKey(void)
```

## Parameters
None.

## Return value
The last pressed key or 0 if not available.

## Description
Returns the last pressed key, useful for custom key mapping.

## Version changes
- EDGELIB 2.70: Added this function to the SDK
