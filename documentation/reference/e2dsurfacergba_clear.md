# Clear


```c++
void Clear(COLORREF col = ECOLOR_BLACK)
```

## Parameters
### col (in)
Color to clear the surface with.

## Return value
None.

## Description
Clear surface contents. Calculated tables will be erased.

## Version changes
- EDGELIB 2.40: Added this function to the SDK
