# OnKeyUp


```c++
void OnKeyUp(unsigned long nr)
```

## Parameters
### nr (in)
The key scancode.

## Return value
None.

## Description
For input objects not part of [ECD](ecd.md): call this function manually after a key has been released.

## Version changes
- EDGELIB 2.70: Added this function to the SDK
