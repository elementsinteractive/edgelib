# Clear


```c++
void Clear(bool colorbuffer = true, bool depthbuffer = true)
```

## Parameters
### colorbuffer (in)
If enabled, clears the screen to a new color.

### depthbuffer (in)
If enabled, clears the depthbuffer. OpenGL only.

## Return value
None.

## Description
Clears the screen.

## Version changes
- EDGELIB 3.96: Added this function to the SDK
